## 1. 에이전트 모드 (Agent Mode)

### 화면 구조

```
┌─────────────────────────────────────────────────┐
│ ← Agent Mode                          [Reset]   │
├─────────────────────────────────────────────────┤
│                                                  │
│  ┌─────────────────────────────────────────────┐│
│  │ 🔍 Thinking...                              ││
│  │ ┌───────────────────────────────────────┐   ││
│  │ │ 📎 web_search("React 19 features")   │   ││
│  │ │    → 5 results found                  │   ││
│  │ └───────────────────────────────────────┘   ││
│  │ ┌───────────────────────────────────────┐   ││
│  │ │ 📎 fetch_url("https://react.dev/...") │   ││
│  │ │    → 2,340 chars extracted            │   ││
│  │ └───────────────────────────────────────┘   ││
│  │                                              ││
│  │ Based on my research, React 19 introduces...││
│  └─────────────────────────────────────────────┘│
│                                                  │
├─────────────────────────────────────────────────┤
│ [🔧 Tools: 5]  [Max Steps: 5]                  │
│ ┌───────────────────────────────────────────┐   │
│ │ Ask anything... Agent will use tools...    │   │
│ └───────────────────────────────────────────┘   │
│                                        [Send]   │
└─────────────────────────────────────────────────┘
```

### 컴포넌트 분해

- **AgentPage** — 메인 뷰 (ViewState: `'agent'`)
- **AgentStepCard** — 각 도구 호출 스텝 (아코디언)
  - `thinking` — 사고 과정
  - `tool_call` — 도구 호출 (이름 + 인자)
  - `tool_result` — 도구 결과 (접이식)
  - `response` — 최종 응답
- **AgentToolConfig** — 활성 도구 선택, max steps 설정

### 사용 가능 도구 (5개)

1. `web_search` — DuckDuckGo 검색 (Modal 프록시)
2. `fetch_url` — URL 내용 가져오기 (Modal 프록시)
3. `calculate` — 수학 계산 (안전한 파서, eval 없음)
4. `get_datetime` — 현재 날짜/시간
5. `analyze_pdf` — 업로드된 PDF 분석 (기존 pdf-extractor 재사용)
